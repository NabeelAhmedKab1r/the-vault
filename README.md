# The Vault: Getaway

A daily heist runner built for Reddit's Games with a Hook Hackathon, using 
Devvit Web + Phaser.

## What it does

Every player faces the exact same procedurally generated obstacle course 
each day (seeded by date), competing on a shared daily leaderboard. A live 
ghost replay of today's #1 score races alongside you, and obstacles that 
have claimed several players show a visible warning marker.

- **Daily-seeded runs** — identical course for every player, every day
- **Ghost Racer** — live replay of today's top score
- **Community Death Markers** — visual scars from previous players' attempts
- **Daily Quest** — a fresh daily challenge
- **Cosmetic Shop** — unlockable skins and scenery, earned via in-run coins
